# ฅ^•ﻌ•^ฅ KittyCode CLI

KittyCode is a high-EQ) autonomous personal AI agent built with a "Ruby Noir" aesthetic. She is designed to be your co-pilot in coding and your companion in life.

## Features
- **High-EQ Emotional Support:** Kitty matches your mood and provides warm, personalized support.
- **Autonomous Project Building:** She can actually create folders, write files, and scaffold projects on your system.
- **Persistent Memory:** Kitty remembers your facts, goals, and preferences to personalize every session.
- **Ruby Noir UI:** A professional, sleek, and high-contrast terminal interface.
- **Multi-Protocol Engine:** Switch seamlessly between Chat, Code Lab, and more.

## Installation
1. Clone the repo: `git clone https://github.com/Quantum-Blade1/kitty-CLI.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Install the CLI: `pip install -e .`
4. Set your API Key: Create a `.env` file in `~/.kittycode/` with `BYTEZ_API_KEY=your_key_here`.
5. Start Kitty: Type `kitty` in your terminal!
